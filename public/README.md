# Vector Tile Map

This is to provide the water supply network data through Mapbox Vector Tile format which is owned by `Narok Water and Sewerage Services Co., Ltd.` in Kenya.

## Usage

Vector tile URL
```
https://narwassco.github.io/vt/tiles/{z}/{x}/{y}.mvt
```
